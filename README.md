Экзаменационное задание по C++
Время выполнения: 90 минут (1.5 часа)
Максимальный балл: 100
Общая постановка задачи
Вам необходимо разработать программу для учета сотрудников и их отпусков в
небольшой компании. Программа должна позволять добавлять сотрудников, управлять их
данными и рассчитывать различные показатели.
Базовый уровень (до 60 баллов)
Часть 1: Класс Employee и основы (30 баллов)

1. Создайте класс Employee (15 баллов):
* Закрытые поля (private): name (строка - имя сотрудника), id (целое число -
уникальный идентификатор), salary (вещественное число - зарплата).
* Публичные методы (public):
* Конструктор, инициализирующий все поля.
* Геттеры (методы получения значений) для всех полей: getName(), getId(),
getSalary().
* Сеттер (метод установки значения) для поля salary: setSalary(double newSalary).
2. Реализуйте в main() (15 баллов):
* Объявите массив (или std::vector) для хранения объектов Employee. Размер массива
- 5.
* В цикле запросите у пользователя данные (имя, ID, зарплата) для 3 сотрудников и
заполните массив созданными объектами.
* Выведите на экран список всех сотрудников в формате: ID: [id], Name: [name], Salary:
[salary].

5. Создайте класс Manager, унаследованный от Employee (15 баллов):
* Добавьте закрытое поле: department (строка - название отдела).
* Реализуйте конструктор, который принимает все параметры родителя и новый
параметр department.
* Переопределите метод getSalary(). Новая зарплата менеджера рассчитывается как:
базовая_зарплата * 1.1 (надбавка 10%).
* Добавьте геттер для поля department.
6. Демонстрация полиморфизма (5 баллов):
* В main() создайте массив указателей на Employee (размером 4).

класс. Для менеджеров должна выводиться зарплата с надбавкой.
